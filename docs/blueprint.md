# **App Name**: Eternal Echo

## Core Features:

- Personalized Setup: Allow users to input their names, partner's name, and a custom message to personalize the proposal.
- Photo Memories: Enable users to upload multiple images to create a personalized gallery of memories.
- Interactive Decision: Provide interactive 'Yes' and 'No' buttons with animations and effects.
- Dynamic Button Behavior: The 'No' button attempts to evade clicks, teleporting elsewhere on the screen. Use reasoning about screen boundaries, user cursor proximity and previous teleport history to improve button behavior. Use memory about recent decisions to discourage button behaviors that frustrate users. This feature uses a tool to help the AI make the best decision.
- Photo Rally: After a 'Yes' response, display a photo rally of uploaded images with a reveal effect.
- Message Display: Show a custom message with the revealed image during the photo rally.
- Animation Effects: Incorporate fade-in, zoom-in, and slide-in animations to enhance the user experience.

## Style Guidelines:

- Primary color: Soft Lavender (#E6E6FA) to evoke a sense of calm and romance.
- Secondary color: Creamy Peach (#FFF0E6) for warmth and tenderness.
- Accent color: Rose Quartz (#F7CAC9) for a delicate and loving touch.
- Headline font: 'Playfair Display' serif for headlines and short chunks of text, providing elegance and readability.
- Body font: 'Lora' serif font for body text, ensuring a gentle and pleasant reading experience.
- Use heart-shaped icons, floral elements, and other gentle, romantic symbols.
- Incorporate smooth, gentle transitions and animated elements for user engagement, such as floating petals or softly glowing effects.
